# Kabi Notes Roadmap

I'm building Kabi Notes as a solo developer to create the private, local-first personal knowledge base I always wanted to use. This document serves as my personal accountability board and tracks my progress towards a `v1.0`.

The app is deeply inspired by my experience with [Obsidian](https://obsidian.md/), [Org-mode](https://orgmode.org/), and [Logseq](https://logseq.com/), but I'm building it from the ground up with a focus on simplicity, performance, and privacy. I will be sharing regular updates on my progress and any challenges I encounter along the way.

## Path to Alpha

- ~~**Local-First Foundation** - Local directory mounting and raw Markdown file parsing.~~
- ~~**Project Home** - Secured `kabinotes.com`~~
- ~**Zen Interface** - Clean UI with sidebar navigation, global search, and document properties.~

## Path to Beta

- **Tiptap Block Editor** - A WYSIWYG block editing experience that safely serializes back to clean, standard Markdown without locking the user in.
- **On-Device AI Integration** - Connecting to local Ollama models (like Qwen) for offline summarization and semantic tagging. 100% private, zero cloud APIs.

---

**Note:** This roadmap represents my current technical focus. Features are added here only when I am actively planning to implement them. I will not add speculative features or wishlist items that I am not committed to building in the near term. This is a living document that will evolve as I make progress and learn from the process.
